Setup — ek baar karo
git --version
Check karo Git installed hai ya nahi
git config --global init.defaultBranch main
Default branch ka naam "main" set karo
Naya repo shuru karna ✓ seekh liya
cd folder-name
Us folder mein jao
git init
Git ko bolo "is folder ko track karo"
git remote add origin URL
GitHub repo se connect karo
git remote -v
Check karo connection sahi hai ya nahi
git remote set-url origin URL
Purana connection badlo naye URL se
Rozana ka kaam ✓ seekh liya
git status
Dekho kya kya change hua hai
git add .
Saari files tayyar karo snapshot ke liye
git add filename
Sirf ek specific file tayyar karo
git commit -m "message"
Snapshot lo — message mein likho kya kiya
git push -u origin main
GitHub par upload karo (pehli baar)
git push
GitHub par upload karo (baad mein)
Team ke saath kaam aage seekhna hai
git pull
GitHub se nayi changes apne laptop par lao
git clone URL
Kisi ka poora repo apne laptop par copy karo
Branching aage seekhna hai
git branch branch-name
Nai branch banao
git checkout -b branch-name
Nai branch banao aur us mein chale jao
git checkout main
Wapas main branch par aao
git merge branch-name
Branch ka kaam main mein mila lo
Open source aage seekhna hai
git fork
Kisi ka repo apne GitHub account mein copy karo
git pull request
Apna kaam original repo mein bhejo review ke liye

.gitignore

README.md
